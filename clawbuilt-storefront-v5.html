Default

Hermes Agent

Built by Nous Research -- the lab behind Hermes models

Self-improving memory that learns your business over time. Five sandboxing backends. Model-agnostic -- route different tasks to different AI models for cost and performance optimization.

  • Persistent memory across sessions
  • Docker sandbox with read-only root, namespace isolation
  • 200+ AI models via OpenRouter
  • Subagent delegation for parallel workflows
  • Telegram, Slack, WhatsApp, Discord, CLI

NanoClaw

~4,000 lines of code -- small enough to audit in an afternoon

Container-per-group isolation for the strongest security boundary. Built on Anthropic's Claude SDK. Our pick for dental, legal, insurance, and veterinary verticals where compliance teams need a clean audit story.

  • OS-level container isolation per agent
  • Fully auditable minimal codebase
  • Claude-powered (Anthropic SDK)
  • WhatsApp, Telegram, Discord, Slack
  • Endorsed by Andrej Karpathy

OpenClaw

210K+ GitHub stars -- the largest agent ecosystem

Maximum integration breadth with 50+ messaging platforms and 5,000+ community skills. We deploy it hardened -- Docker sandbox, firewall rules, credential isolation. Best when you need a niche integration no other framework supports.

  • 50+ platform integrations
  • 5,000+ community skills via ClawHub
  • Deployed with full security hardening
  • Any AI model via config
  • Largest community and ecosystem

FAQ

Questions we get
from business owners

Real objections, straight answers.

Will the AI give my customers wrong information? +
Your config ships with a knowledge base scaffold you fill with your own data -- services, prices, policies. The agent answers from that, not from general knowledge. For healthcare and legal verticals, compliance guardrails are built into the system prompt to prevent the agent from making claims outside its lane.
What if it books an appointment we can't fill? +
We configure the agent with read-only calendar access by default. It redirects to your booking link or captures a lead -- it doesn't directly write to your calendar until you've explicitly expanded that permission and tested it. That's the "build trust before expanding access" principle we follow on every deployment.
Can it access my QuickBooks or financial data? +
Only if you explicitly wire that integration, which we'd flag as a high-permission step. Each vertical config has a defined integration boundary. A DentalClaw instance won't touch your accounting tools because they're not in scope. Composio OAuth also means the agent only gets access to what you specifically authorized -- nothing more.
Is this HIPAA safe for my dental or medical practice? +
DentalClaw and PawClaw are built with PHI firewalls in the system prompt -- the agent won't collect, store, or repeat protected health information in its responses. That said, HIPAA compliance is your responsibility as a covered entity. We build in the technical safeguards; you should confirm with your privacy officer or counsel before processing PHI of any kind.
Do I need technical knowledge to use a config I bought? +
The Starter config requires some comfort with a terminal and following a setup guide. Most tech-comfortable business owners or their office managers can handle it. If you'd rather not touch a command line, that's exactly what the Done Install is for -- we handle everything remotely.
What happens when something breaks at 2am? +
On a retainer plan, we monitor your deployment's health and typically catch issues before they surface. The Command plan includes a 4-hour SLA response. Block hours can also be used for emergency troubleshooting. If you're self-hosted with no support plan, the agent will just stop responding until you investigate -- it won't do anything harmful, just go quiet.
Which framework do you deploy on? +
We support three: Hermes Agent (our default for most verticals -- self-improving memory, five sandboxing backends, 200+ model support), NanoClaw (our pick for regulated industries like dental, legal, and insurance -- minimal codebase, container-per-group isolation), and OpenClaw (when you need a niche integration from their 5,000+ skill ecosystem). We recommend the best fit during your scoping call.

Yes. We offer a Security Audit ($997) to review your existing deployment and close gaps, or a full Harness Migration ($1,497) to move you to Hermes Agent or NanoClaw with hardened security, config translation, and 14-day ActiveCare support. Most migrations are completed same-day.
Can't I just download OpenClaw and set it up myself? +
You can -- OpenClaw, Hermes Agent, and NanoClaw are all open source. But a raw install without vertical tuning is a blank slate. OpenClaw specifically has documented security issues (the ClawHavoc attack compromised 9,000+ installations). Our configs give you industry-specific tuning that would take weeks to build yourself, deployed on a hardened framework matched to your compliance needs.
